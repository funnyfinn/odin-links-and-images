Character Sets

You can use UTF-8 to count in Turkish:

  1. Bir
  2. İki
  3. Üç
  4. Dört
  5. Beş

HTML Entities

There are three reserv characters in HTML:

  • & < >

You should always use HTML entities for these characters to ensure proper rendering.

If you’re into “web typography,” you’ll also find yourself using curly quotes quite a bit.
